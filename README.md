# 开放世界ARPG原型

一个受《原神》、《鸣潮》等启发，使用UE5、C++和GAS开发的个人项目。（不含二进制项目文件）



## 功能列表

- 基于 Gameplay Ability System (GAS) 的战斗系统 (连击、技能、闪避)
- 基于行为树的敌人 AI (巡逻、追击、攻击)
- 支持队伍角色切换
- UMG 和 Widget Component 实现的动态血条



## 版本

- Unreal Engine 5.2,

- Visual Studio 2022, MSVC 14.34




## **项目依赖**

- KawaiiPhysics
- SPCRJointDynamics
- UnLua
